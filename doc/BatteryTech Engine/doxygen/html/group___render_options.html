Detailed Description

Setting options for render items.

Once a render item is created via any render call, such as Game::render2D() or Game::renderAssimp(), the render item can be modified with a number of options.

Not all render items support all parameters. For example, alignment is only supported for text rendering and alpha/color parameters work on all but text rendering. Parameter List:

align Sets text alignment

  • left Left Alignment
  • center Centered Alignment
  • right Right Alignment

alpha Sets the alpha color

  • The alpha value 0-1

colorFilter Sets the full 4 component color

  • r,g,b,a The red,green,blue,alpha values 0-1

nofog Disables fog (if globally enabled)

  • true or false

maxPointLights Defines the maximum number of point lights that will be used simultaneously for this object

  • the maximum number (default is 3)

useCameraClosestPointLights Normally the point lights closest to the object are used - this uses the ones closest to the camera

  • true or false

viewport Sets an alternate viewport to use

  • x,y,width,height in screen coordinates

nodesCullFrustum Enables frustum culling of nodes in scene

  • true or false

nodirlight Disabled global/directional light if enabled

  • true or false

noshadowgen Disables shadowmap generation if enabled

  • true or false

noshadowrecv Disabled receiving of shadows if enabled

  • true or false

drawfirst Forces drawing order for 3D render calls

  • true or false

twosided Enables two-sided drawing

  • true or false

multiline Enables multiline measurement and text rendering

  • true or false

isOpaque Specifies if the item is to be rendered in a translucent or opaque pass

  • true or false

lightperpixel Specified per-pixel lighting (false is per-vertex, default)

  • true or false
