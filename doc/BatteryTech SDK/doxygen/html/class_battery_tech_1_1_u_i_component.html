User Interface Library

A User Interface ComponentThis is the foundation and base class of all User Interface Components in the BatteryTech UI Library. More...

Detailed Description

UIComponents may be independent, they may be layouts or they may have any number of child components.

#include <batterytech/ui/UIComponent.h>
