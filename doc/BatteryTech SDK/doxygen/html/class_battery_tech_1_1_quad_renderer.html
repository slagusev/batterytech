void BatteryTech::QuadRenderer::startBatch ( )

A batch mode render is one where as many quads are drawn in one shot as possible. Switching textures forces a draw so to optimize, use texture atlases. Batch mode requires VBO support (available on most devices and platforms) and will revert to regular rendering if not available)
