void BatteryTech::ShaderProgram::bind ( )

We have a very optimized bind here.

1) It won't rebind to a shader that's already bound 2) It has optimizations for sequentially numbered vertex attribute locations starting at 0 2a) In this case, which is either 99 or 100% of the time, this will only unbind/bind to deltas in the attribute counts.
