BatteryTech::GameConnection Class Reference

Manages an active game connection in either host or client mode. More...

Detailed Description

Once the network manager has made a connection, it will create a GameConnection to the other side. This GameConnection is what is used to transmit and receive data from a specific client or to the server. Normally you don't need to do anything with this class because NetworkManager will use it internally to marshall data and manage the connection.
