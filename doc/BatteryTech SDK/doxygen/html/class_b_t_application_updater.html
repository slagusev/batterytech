BTApplicationUpdater Class Reference
Application Entry Points

Entry point for updating a BatteryTech application. More...

Detailed Description

In a normal state-driven application, update is where the processing logic should be. It is a good practice to keep all rendering code separate, but of course calculations needed to drive rendering later can be fine in phase.

This class needs to be extended and added to an extended Context to provide update capabilities.
