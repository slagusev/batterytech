Primitive Types

BatteryTech defines primitive types for cross-platform compatibility. More...

Detailed Description

Common typedefs include:
S8, S16 and S32 = Signed 8, 16 and 32 bit integers respectively
U8, U16 and U32 = Unsigned 8, 16 and 32 bit integers respectively
BOOL32 for booleans with TRUE and FALSE as values
F32 and F64 = 32 bit and 64 bit floating point number

Example:

BOOL32 found = FALSE;
for (S32 i = 0; i < numObjects; i++) {
if (matches(objects[i])) {
found = TRUE;
break;
}
}
