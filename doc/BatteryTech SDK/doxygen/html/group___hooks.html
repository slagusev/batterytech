All hooks require code on the platform.

Parameters:
hookThe hook data
resultThe result (if the hook is synchronous, otherwise Context::callbackData will hold the result)
resultLenThe length of the result buffer

Example:

char resultBuffer[10];
_platform_hook("hasFrontCamera", resultBuffer, 10);
if (strEquals(resultBuffer, "true")) {
logmsg("has front camera");
}
